# مشغل الفيديو (MyVideoPlayer)

تطبيق أندرويد كامل لتشغيل الفيديوهات المخزّنة على الجهاز، مبني بلغة Kotlin ومكتبة Media3 (ExoPlayer).

## المميزات
- عرض كل فيديوهات الجهاز تلقائياً مع صورة مصغّرة (thumbnail) لكل فيديو، الاسم، الحجم، والمدة.
- طلب صلاحيات الوصول للفيديوهات تلقائياً (متوافق مع أندرويد 13+ وما دونه).
- شاشة تشغيل مخصّصة بالكامل:
  - تشغيل / إيقاف مؤقت.
  - تقديم وترجيع 10 ثواني بالأزرار أو بالنقر المزدوج (يسار الشاشة = ترجيع، يمين الشاشة = تقديم، منتصف = تشغيل/إيقاف).
  - السحب رأسياً في **النصف الأيسر** من الشاشة للتحكم بـ **السطوع**.
  - السحب رأسياً في **النصف الأيمن** من الشاشة للتحكم بـ **مستوى الصوت**.
  - زر تغيير سرعة التشغيل (0.5x, 1x, 1.25x, 1.5x, 2x).
  - أزرار التالي/السابق للتنقل بين فيديوهات القائمة (playlist).
  - زر قفل الشاشة لمنع اللمس العرضي أثناء المشاهدة.
  - إخفاء تلقائي لعناصر التحكم بعد 3.5 ثانية، وإظهارها بالنقر.
  - وضع ملء الشاشة الغامر (إخفاء أشرطة النظام).
  - شريط تقدم (Seek Bar) مع عرض الوقت الحالي والكلي.

## كيفية فتح المشروع وتشغيله
1. افتح **Android Studio** (يفضّل أحدث إصدار، Hedgehog أو أحدث).
2. اختر **Open** ثم حدد مجلد المشروع `MyVideoPlayer`.
3. انتظر حتى ينتهي Android Studio من مزامنة Gradle تلقائياً (سيقوم بتنزيل Gradle Wrapper وكل الاعتماديات المطلوبة من الإنترنت أول مرة).
4. وصّل جهاز أندرويد حقيقي أو شغّل محاكي (Emulator).
5. اضغط **Run ▶** لتثبيت التطبيق وتشغيله.

> ملاحظة: عند أول تشغيل سيطلب منك التطبيق صلاحية الوصول للفيديوهات — اقبلها لتظهر لك قائمة الفيديوهات.

## هيكل المشروع
```
MyVideoPlayer/
├── app/
│   ├── build.gradle.kts          # اعتماديات التطبيق (Media3, Glide, AndroidX...)
│   └── src/main/
│       ├── AndroidManifest.xml
│       ├── java/com/example/myvideoplayer/
│       │   ├── MainActivity.kt       # الشاشة الرئيسية (قائمة الفيديوهات)
│       │   ├── PlayerActivity.kt     # شاشة التشغيل الكاملة
│       │   ├── VideoAdapter.kt       # Adapter لعرض القائمة
│       │   ├── VideoRepository.kt    # جلب الفيديوهات من MediaStore
│       │   └── model/VideoItem.kt    # نموذج بيانات الفيديو
│       └── res/
│           ├── layout/                # activity_main, activity_player, item_video
│           ├── drawable/               # أيقونات vector لكل الأزرار
│           └── values/                 # الألوان، النصوص، الثيمات
├── build.gradle.kts
└── settings.gradle.kts
```

## بناء التطبيق من الموبايل فقط (بدون كمبيوتر)
المشروع يحتوي على ملف بناء تلقائي جاهز في `.github/workflows/build-apk.yml`. اتبع الخطوات بالمتصفح من الموبايل:

1. افتح **github.com** وسوّي حساب (لو ما عندك).
2. اضغط **+** أعلى الصفحة > **New repository**، عطيه اسم مثل `MyVideoPlayer`، خله **Public** أو Private، ثم **Create repository**.
3. بنفس صفحة الريبو الفاضية، اضغط رابط **uploading an existing file**.
4. فك ضغط ملف `MyVideoPlayer.zip` أول شي (استخدم أي تطبيق فك ضغط زي ZArchiver)، بعدين من صفحة الرفع بالمتصفح اختر **كل الملفات والمجلدات** بالمرة وارفعها. تأكد المجلدات مثل `app` و `.github` انرفعت وياها.
5. اكتب رسالة بسيطة بالأسفل مثل "رفع المشروع" واضغط **Commit changes**.
6. روح لتبويب **Actions** أعلى صفحة الريبو — راح تلقى عملية بناء (Build APK) شغالة تلقائياً. انتظر شوي لين تصير علامة ✅ خضراء.
7. اضغط على العملية المكتملة، وبالأسفل بقسم **Artifacts** راح تلقى ملف اسمه `MyVideoPlayer-apk` — حمّله.
8. فك ضغط الملف المُنزّل، بتلقى `app-debug.apk`. اضغط عليه لتثبيته (لازم تفعّل خيار "السماح بالتثبيت من مصادر غير معروفة" أول مرة).

## تخصيص اسم الحزمة (Package Name)
اسم الحزمة الحالي هو `com.example.myvideoplayer`. لتغييره، استخدم خاصية **Refactor > Rename Package** داخل Android Studio بدل التعديل اليدوي حتى تنعكس التغييرات على كل الملفات.

## ملاحظات تقنية
- الحد الأدنى لإصدار أندرويد المدعوم: **Android 7.0 (API 24)**.
- الإصدار المستهدف: **Android 14 (API 34)**.
- المشغّل يعتمد على مكتبة **androidx.media3 (ExoPlayer)** الرسمية من Google، وهي بديل مكتبة ExoPlayer القديمة وتُنصح بها حالياً لتشغيل الفيديو/الصوت.
- صور الفيديوهات المصغّرة تُحمّل عبر مكتبة **Glide** التي تدعم توليد thumbnail مباشرة من روابط الفيديو (content URI).
